# 百货网站

这是一个电子商务百货网站系统，提供各类商品的在线购物体验。

## 功能特点

- 用户注册与登录
- 商品分类浏览
- 商品搜索
- 购物车管理
- 订单处理
- 用户评价系统
- 个人中心管理

## 技术栈

- 前端：HTML5, CSS3, JavaScript, Vue.js
- 后端：Node.js, Express
- 数据库：MongoDB

## 如何运行

1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 运行开发服务器：`npm run dev`
4. 在浏览器中访问：`http://localhost:3000`
